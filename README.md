# Xenos Services

Premium digital solutions for gaming, streaming, and security.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📦 Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lenis Smooth Scroll
- Lucide Icons

## 🌐 Deployment

This project is configured for Vercel deployment.

```bash
npm run deploy
```
